Css

CSS (Cascading Style Sheet) the HTML elements which are displayed on screen, paper, or in other media. It saves a lot of time. It controls the layout of multiple web pages at one time. It sets the font-family, color, background color on the page.>

There are 3 types of CSS which are below:

  • Inline CSS
  • Internal CSS
  • External Css

Inline Css

Inline Css is a type of css in which the propert are written in front of the attribute itself.

Inline Css has the highest Priority among the other two Css types.

Internal Css Css:

The Internal CSS has tag in the "section" of the document. This CSS "style" is an effective way to "style" single pages. Using the CSS "style" for multiple web pages is time-consuming because we require placing the style on each web page.

Internal Css has Second highest priority over the other Css type.

External Css:

In external CSS, we link the web pages to the external .css file. It is created by text editor. The CSS is more efficient method for styling a website. By editing the .css file, we can change the whole site at once.

It has slightly low priority has compared to other types.
